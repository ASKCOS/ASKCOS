makeit.utilities.io.draw.CheckAtomForGeneralization(atom)[source]¶

Given an RDKit atom, this function determines if that atom’s SMART representation was likely a result of generalization. This assumes that the transform string was generated using explicit Hs with aliphatic carbons as C, aromatic carbons as c, and non-carbons as #N where N is the atomic number of the generalized species.
