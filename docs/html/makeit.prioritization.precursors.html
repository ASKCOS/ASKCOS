Bases: makeit.prioritization.prioritizer.Prioritizer

Standalone, importable SCScorecorer model.

It does not have tensorflow as a dependency and is a more attractive option for deployment. The calculations are fast enough that there is no real reason to use GPUs (via tf) instead of CPUs (via np).
