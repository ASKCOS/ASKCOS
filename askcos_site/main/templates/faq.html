Why do I get an error message when trying to run an expansion?

Because we are running these services locally at MIT, there are a limited number of workers available to perform cmoputationally-intensive tasks. There is a chance that other users were running expansions at the same time, so no available pool of workers was found. In rare cases, these workers may run into an error state and refuse to accept incoming requests, at which time they will be restarted. If you run into any issues, reach out using the Comment link on the upper navigation bar and we will look into the issue ASAP.
